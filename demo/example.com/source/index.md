Welcome to Example.com. This site exercises every feature of the jekyll-aeo gem.

## Features Under Test

- Markdown page generation
- llms.txt and llms-full.txt
- Link tag injection
- JSON-LD schemas (FAQPage, HowTo, BreadcrumbList, Organization, Speakable, Article)
- robots.txt generation
- domain-profile.json
- URL map generation
