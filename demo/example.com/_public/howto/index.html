How to Install jekyll-aeo

How to Install jekyll-aeo

Follow these steps to get started.

Step 1: Add the gem

Add the following to your Gemfile:

gem "jekyll-aeo"

Step 2: Install dependencies

Run bundle install.

Step 3: Configure

Add settings to _config.yml.

Step 4: Build

Run jekyll build and verify the output.
